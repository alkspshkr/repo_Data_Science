# Проект - Анализ рынка недвижимости

## Описание проекта

Используя данные сервиса Недвижимости, определить рыночную стоимость объектов недвижимости и типичные параметры квартир

## Результаты проекта

На основе полученных данных от сервиса недвижимости, провели знакомство с данными, определелили наиболее значимые факторы, влияющие на стоимость недвижимости в городе и регионе, рассмотрели типичные параметры квартир, оценили аномальные значения в сроках жизни объявления, проанализировали распредления параметров недвижимости в зависимости от удаленности от центра города, Выделили ТОП-10 пунктов по размещенным объявлениям. Для бизнеса построили необходимые графики: гистограммы распределения, диаграммы размаха, столбчатые диаграммы, точеченый график.

## Используемый стэк:

`Pandas`, `Python`, `Matplotlib`, `Seaborn`
